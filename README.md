# Predictive-Maintenance-of-Aircraft-Engines-using-Machine-Learning
Predictive Maintenance of Aircraft Engines using Convolutional Neural Network and Random Forest displayed on a dashboard using Streamlit.
A dual system model where Convolutional Neural Network (CNN) predicts the Remaining Useful Life (RUL) of aircraft engines and the ones with low RUL are classified as critical and good by Random Forest (RF) model.
